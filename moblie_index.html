⚑ Support & Feedback
Having issues or a feature request? We're here to help.
Quick Help
🔒
Embed Restricted Video
Some YouTube videos disable embedding. Use "Try Nocookie Mode" or "Watch on YouTube" buttons that appear automatically when a video can't be embedded.
⏱
Slow First Load (Cold Start)
FREQ runs on Render.com. Enable Keep Server Awake (in Account → Keep-Alive) to prevent 30-60s cold start delays. It pings the server every 4 minutes automatically.
🎵
Local File EQ / Effects Not Working
EQ, reverb, and effects only apply to local files played through FREQ's Web Audio engine. Streaming embeds (YouTube, Spotify, etc.) use their own players.
📋
Playlist Sync Not Working
Sign in to your FREQ account, then use ↑ Push to Server in the Account panel. Playlists sync automatically every 10 minutes when Keep-Alive is on.
Send Bug Report or Feedback
Select a category, describe your issue, then send. Your report will be emailed directly to the FREQ developer.
